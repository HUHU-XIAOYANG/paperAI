## ✨ 主要功能

### 核心功能

- ✅ **多智能体协作系统**: 支持 20+ 并发智能体同时工作
- ✅ **智能任务分配**: Decision AI 自动分配写作任务
- ✅ **实时协作可视化**: 动态团队可视化器展示智能体协作状态
- ✅ **智能审稿机制**: 多层级审稿流程（同行评审、副主编、主编）
- ✅ **修订历史**: 完整的工作历史和修订记录
- ✅ **文档导出**: 支持 Markdown、Word、PDF 格式导出

### 智能体角色

1. **Decision AI** - 决策智能体，负责任务分配和流程控制
2. **Supervisor AI** - 监管智能体，监控整体进度和质量
3. **Writer** - 写作智能体，负责论文内容撰写
4. **Peer Reviewer** - 同行评审专家，提供专业评审意见
5. **Deputy Editor** - 副主编，进行二次审核
6. **Editor in Chief** - 主编，最终审核和决策
7. **Editorial Office** - 编辑部，协调流程和沟通

### 高级特性

- 🔒 **配置加密**: API Key 等敏感信息加密存储
- 🌐 **网络权限管理**: 智能体网络查询权限控制
- 📊 **性能监控**: 实时性能指标监控
- 🎨 **玻璃态设计**: 现代化的 UI 设计风格
- 🌓 **主题切换**: 支持亮色/暗色主题

## 📝 使用指南

### 首次配置

1. **启动程序**
   - 双击 `AgentSwarmWritingSystem.exe`
   - 等待程序加载完成

2. **配置 AI 服务**
   - 进入配置界面
   - 选择 AI 服务类型（OpenAI 或兼容服务）
   - 输入 API Key
   - 输入 Base URL（如果使用自定义服务）
   - 点击"测试连接"验证配置

3. **填写论文信息**
   - 输入论文主题
   - 填写研究领域
   - 设置论文要求和规范
   - 选择需要的智能体角色

4. **开始写作**
   - 点击"开始写作"按钮
   - 观察智能体协作过程
   - 实时查看写作进度

### 常用操作

- **查看智能体状态**: 在动态团队可视化器中查看
- **查看交互历史**: 在交互时间线中查看
- **查看工作输出**: 在工作展示面板中查看
- **导出论文**: 点击导出按钮，选择格式
- **查看修订历史**: 在修订历史面板中查看

## 🔍 故障排除

### 常见问题

#### 问题 1: 程序无法启动

**可能原因**:
- Windows 版本不兼容
- 杀毒软件拦截
- 缺少必要的系统组件

**解决方案**:
1. 确保使用 Windows 10/11 64位系统
2. 将程序添加到杀毒软件白名单
3. 尝试以管理员身份运行
4. 检查 Windows 更新

#### 问题 2: 找不到 prompts 文件夹

**可能原因**:
- prompts 文件夹被移动或删除
- 文件夹权限问题

**解决方案**:
1. 确保 prompts 文件夹与 .exe 文件在同一目录
2. 重新解压发布包
3. 检查文件夹权限

#### 问题 3: AI 服务连接失败

**可能原因**:
- API Key 错误
- Base URL 配置错误
- 网络连接问题
- API 服务不可用

**解决方案**:
1. 验证 API Key 是否正确
2. 检查 Base URL 格式（应包含 http:// 或 https://）
3. 测试网络连接
4. 检查 API 服务状态

#### 问题 4: 界面显示异常

**可能原因**:
- 缓存问题
- 渲染错误
- 主题设置问题

**解决方案**:
1. 重启程序
2. 切换主题（亮色/暗色）
3. 检查错误日志
4. 清除应用数据（谨慎操作）
