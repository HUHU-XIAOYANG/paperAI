## 使用说明

### 配置 AI 服务
1. 点击"配置"按钮
2. 选择 AI 服务类型
3. 输入 API Key 和 Base URL
4. 测试连接
5. 保存

### 开始写作
1. 输入写作主题
2. 点击"开始写作"
3. 观察智能体协作
4. 查看写作结果

## 常见问题

**Q: 程序无法启动？**  
A: 检查杀毒软件是否拦截，添加到白名单

**Q: 无法连接 AI 服务？**  
A: 检查 API Key、Base URL 和网络连接

**Q: 点击"开始写作"没反应？**  
A: 确保已配置 AI 服务并输入主题

**Q: 仍然出现白屏？**  
A: 确保 prompts 文件夹存在，重启程序
