# Agent Swarm Writing System - Windows版本

## 安装说明

这是一个便携版本，无需安装。

### 运行程序

1. 双击 AgentSwarmWritingSystem.exe 启动程序
2. 首次运行时，程序会在用户目录下创建配置文件夹

### 系统要求

- Windows 10 或更高版本
- 64位操作系统

### 配置文件位置

配置文件保存在：%APPDATA%\com.agentswarm.writingsystem\

### 提示词模板

提示词模板位于 prompts 目录中，您可以根据需要自定义。

## 功能特性

- 多Agent协作写作
- 实时流式输出
- 动态团队可视化
- 文档导出（DOCX、Markdown、PDF）
- 联网搜索功能
- 退稿机制和修订历史

## 技术支持

如有问题，请查看项目文档或提交Issue。
