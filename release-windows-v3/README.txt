重要修复
================================================================================

本版本修复了关键的 React Error #185 问题：

✅ 修复了 DynamicTeamVisualizer 组件的无限循环问题
   - 问题：agentsMap 每次都是新引用，导致 useMemo 和 useEffect 无限触发
   - 修复：直接使用 getAllAgents() 方法，避免 Map 对象引用问题
   - 添加了 calculatePositions 到 useEffect 依赖数组

✅ 更新了 Vite 配置，排除测试文件
   - 确保测试文件不会被打包到生产版本中

使用方法
================================================================================

【方式一：便携版（推荐）】

1. 直接双击运行 AgentSwarmWritingSystem.exe
2. 首次运行会自动创建配置文件
3. 无需安装，可以放在任意目录运行
4. 确保 prompts 文件夹与可执行文件在同一目录

【方式二：安装版】

1. 双击运行 Agent Swarm Writing System_0.1.0_x64-setup.exe
2. 按照安装向导完成安装
3. 从开始菜单或桌面快捷方式启动程序

================================================================================
系统要求
================================================================================

- 操作系统：Windows 10/11 (64位)
- 内存：建议 4GB 以上
- 磁盘空间：至少 200MB 可用空间
- 网络：需要互联网连接以访问 AI 服务

================================================================================
首次使用配置
================================================================================

1. 启动程序后，进入配置界面
2. 配置 AI 服务：
   - OpenAI API：输入 API Key 和 Base URL
   - 或其他兼容 OpenAI 格式的 AI 服务
3. 填写论文主题和要求
4. 点击"开始写作"启动智能体协作

故障排除
================================================================================

【问题：程序无法启动】
- 确保 Windows 10/11 64位系统
- 检查是否被杀毒软件拦截
- 尝试以管理员身份运行

【问题：找不到 prompts 文件夹】
- 确保 prompts 文件夹与 .exe 文件在同一目录
- 不要移动或删除 prompts 文件夹

【问题：AI 服务连接失败】
- 检查网络连接
- 验证 API Key 是否正确
- 确认 Base URL 格式正确

【问题：界面显示异常或卡顿】
- 重启程序
- 检查是否有错误提示
- 本版本已修复无限循环问题
