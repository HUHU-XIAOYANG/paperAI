# Components

This directory contains React UI components:

## Core Components
- **GlassMorphism**: Glass morphism container component
- **WorkDisplayPanel**: Individual agent work display
- **InteractionTimeline**: Message history timeline
- **DynamicTeamVisualizer**: Team structure visualization

## Layout Components
- **MainLayout**: Main application layout
- **NavigationBar**: Top navigation bar
- **ConfigPanel**: Configuration interface

## Feature Components
- **TopicInput**: Paper topic input form
- **AgentCard**: Agent information card
- **MessageBubble**: Chat message display
- **StatusIndicator**: Agent status indicator

Each component will be implemented in subsequent tasks.
